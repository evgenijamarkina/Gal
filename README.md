# GitHub User Finder

**Автор:** Дементьев Виталий Андреевич

**Дата сдачи:** 29.04.2026

##  Краткое описание

GUI-приложение на Python для поиска пользователей GitHub через публичный API. Позволяет находить профили разработчиков по логину, просматривать детальную информацию (аватар, имя, био, количество репозиториев и подписчиков), сохранять понравившихся пользователей в избранное. Данные избранного автоматически сохраняются в JSON-файл и загружаются при следующем запуске.

##  Функциональность

- Поиск пользователей GitHub по логину
- Отображение информации: аватар, имя, описание, статистика
- Добавление пользователей в избранное
- Сохранение избранного в favorites.json
- Быстрый переход в профиль на GitHub
- Валидация ввода (проверка на пустое поле)
- Обработка ошибок API (лимит запросов, пользователь не найден)

##  Требования для запуска

Что нужно установить на компьютер:

- Python 3.10 или выше
- Библиотеки: `pip install requests Pillow`

##  Как запустить
``` bash
git clone https://github.com/W1rr0/GitHub-User-Finder
cd github-user-finder
python main.py
``` 
## Как пользоваться API

Приложение использует публичный GitHub REST API v3 без аутентификации.
```
GET https://api.github.com/users/{username}
```

## Особенности публичного API

- 60 запросов с одного IP без авторизации

- Ошибка "403 с сообщением API rate limit exceeded" при превышении лимита

- Сброс лимита каждый час

- Документация: https://docs.github.com/ru/rest

## Как пользоваться

При запуске появляется главное окно приложения, введите логин GitHub в поле поиска, нажмите кнопку Найти или клавишу Enter, результат поиска отобразится в рамке «Результат поиска»:

**Аватар пользователя**

**Логин и имя**

**Описание профиля (bio)**

**Статистика: репозитории, подписчики, подписки**


Чтобы добавить пользователя в избранное, нажмите "Добавить в избранное", все избранные пользователи отображаются в нижнем списке.
Чтобы открыть профиль в браузере: выберите в списке → Открыть профиль.
Чтобы удалить из избранного: выберите в списке → Удалить.
